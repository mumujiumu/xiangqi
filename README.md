# 中国象棋 · 楚河汉界

一个用纯 HTML + CSS + Canvas JavaScript 实现的中国象棋游戏，开箱即玩，无需任何依赖。

## 在线玩法

直接用浏览器打开 `index.html` 即可开始对弈。

## 功能特性

- **完整棋盘**：9×10 标准棋盘，楚河汉界、九宫格、兵线标记一应俱全
- **全部棋子**：帥仕相馬車炮兵 / 將士象馬車炮卒，32 枚棋子标准布局
- **走法规则**：每种棋子严格遵循象棋规则
  - 帅/将：九宫内一步直行
  - 仕/士：九宫内一步斜行
  - 相/象：田字步、不过河、不塞象眼
  - 马：日字步、蹩马腿
  - 车：直线任意步
  - 炮：移动如车，吃子隔山打牛
  - 兵/卒：过河前只进，过河后可横走
- **将军/将死判定**：自动检测将军、将死、困毙
- **飞将规则**：两将照面自动判负
- **走子限制**：不能走出让己方被将军的棋（送将非法）
- **悔棋功能**：随时撤回上一步
- **翻转棋盘**：随时翻转视角
- **棋谱记录**：右侧实时记录每一步走法
- **俘获展示**：侧边栏展示双方被吃棋子

## 操作说明

1. 点击己方棋子选中（红方先行）
2. 绿点为可走位置，红圈为可吃位置
3. 点击目标位置完成走子
4. 再次点击其他己方棋子可切换选中

## 技术栈

- HTML5 Canvas 绘制棋盘和棋子
- 原生 JavaScript 实现游戏逻辑
- CSS3 界面美化
